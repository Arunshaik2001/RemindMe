# RemindMe
reminder app

# About this app

**CLEAN architecture**

**dependencies**:
1. RoomDB
2. WorkManager
3. Hilt
4. Livedata
5. Kotlin Flow

**This app currenlty has 2 screens**
1. To see list reminders in progress and completed and can delete the reminders by swipping and clicking cross icon.
2. To set reminders.
